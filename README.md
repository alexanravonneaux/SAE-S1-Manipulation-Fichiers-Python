## 📋 Description

Manipulation de différents formats de fichiers (CSV, JSON, XML, TXT) avec Python. Ce projet couvre la lecture, l'écriture, le filtrage et la transformation de données structurées, en utilisant les bibliothèques standards et pandas.

## 🎯 Objectifs pédagogiques
- Lire et écrire des fichiers CSV, JSON, TXT en Python
- Filtrer et trier des données avec pandas et les modules standards
- Transformer des structures de données (dictionnaires, listes, DataFrames)
- Gérer les erreurs et exceptions lors du traitement de fichiers

## 🔧 Technologies

| Outil | Rôle |
|-------|------|
| Python 3 | Langage principal |
| pandas | Manipulation de données tabulaires |
| csv / json | Modules standards de lecture/écriture |
| os / pathlib | Gestion des chemins de fichiers |
